# for-ovtis
Сервер с кнопкой вкл\выкл которую можно менять через адресную строку.

## Как использовать
1. Скачать файлы и распаковать их в одну папку
2. Открыть командную строку
3. Сменить директорию на путь до папки
4. Необходимо установить версию PHP 5.4 или новее.
5. Запустить команду `php -S localhost:8000` (не закрывать командную строку)
## Как сменить статус
* Прописать команду `curl "http://localhost:8000/?status=on"` в командной строке.
* В отдельной вкладке прописать `http://localhost:8000/?status=on`.
* В файле `status.txt` изменить значение on/off.
